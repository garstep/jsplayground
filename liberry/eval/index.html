eval()

"The eval() function evaluates JavaScript code represented as a string... If the string represents an expression, eval() evaluates the expression." - Mozilla.org

What is this?

It sounds like a lot of people think this function is evil, so maybe it shouldn't be used that much.

"eval() is a dangerous function, which executes the code it's passed with the privileges of the caller. If you run eval() with a string that could be affected by a malicious party, you may end up running malicious code on the user's machine with the permissions of your webpage / extension. More importantly, third party code can see the scope in which eval() was invoked, which can lead to possible attacks in ways to which the similar Function is not susceptible." Mozilla.org

Example of Misuse

Mistake

function isChecked(optionNumber) {

return eval("forms[0].option" + optionNumber + ".checked");

}

var result = isChecked(1);

Correct

function isChecked(optionNumber) {

return forms[0]["option" + optionNumber].checked;

}

var result = isChecked(1);
