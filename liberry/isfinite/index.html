isFinite()

The global isFinite() function determines whether the passed value is a finite number. If needed, the parameter is first converted to a number. - Mozilla.org

That's my mom's China. It's fine..aight?

Example Code

Is it Finite?

isFinite(Infinity); // false

isFinite(NaN); // false

isFinite(-Infinity); // false

isFinite(0); // true

isFinite(2e64); // true

isFinite(910); // true

isFinite(null); // true

isFinite("0"); // true, would've been false

with the // more robust Number.isFinite("0")

Real World Application

You're at your friend Ricky's house and he still lives with his mom. He's very paranoid about you breaking his mom's fine China, so he reminds you to be careful around it constantly. Everytime he reminds you, you have to return a string (vocally tell him) "stop worrying about it".

He reminds you about it so many times that you want to figure out how many more times he could possibly ask you. You build a program that uses python machine learning tactics and automatically generates probability equations that will predict the maximum amount of times he could could possibly remind you about the china.

Your robot's not perfect, however, and sometimes it generates an equation that doesn't factor in that at some point, being a human, Ricky will pass away, leaving him no ability to remind you about the china after that time. Some equations predict that based on his current rate of reminders, he will continue on forever. You only want to dig through equations that might be good predictors of the estimate, including Rick'y inability to live forever.

You could use isFinite() inside of a javascript function that will automatically trash any auto-generated equations that predict Ricky will keep talking about the China forever. These would be equations that will predict the number of times he will ask you is infinite. That way, you can astutely spend your time only looking through feasible equations that could help you answer your question. This is a very important task to you wouldn't want to waste any time doing it inefficiently.
